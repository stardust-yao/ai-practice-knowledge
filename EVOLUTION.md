## v0.2 — 2026-07-18｜抓取方案确定 + 目录结构设计

**变更内容：**
- 抓取方案从 we-mp-rss 切换为 **wechat2rss.xlab.app**
- 确定仓库目录结构（原始存档 + 主题分类双轨）

**调研结论（2026-07-18 实测）：**

| 方案 | 状态 | 说明 |
|------|------|------|
| we-mp-rss | ❌ 不可用 | Issue #425：微信服务端改版，所有版本均受影响，二维码 404 |
| wechat_articles_spider | ❌ 排除 | README 明确"没法直接使用"，2023 年后无维护 |
| wechat2rss.xlab.app | ✅ **选定** | 腾讯技术工程已在免费索引，2026-07-17 文章实测可抓，完全自动 |
| careerengine.us | ❌ 已死 | 404 Not Found |
| wx.qq.com 自动化 | ❌ 不可用 | 国外 IP 超时，无法访问 |
| 商业 API（新榜/清博） | ❌ 排除 | ¥2000+/月，成本过高 |

**已验证 RSS Feed：**
```
https://wechat2rss.xlab.app/feed/9685937b45fe9c7a526dbc32e4f24ba879a65b9a.xml
```
- 服务运行自 2021 年，GitHub 1.5k stars，最新提交 2026-06-17
- 腾讯技术工程在免费公共索引中，无需 Cookie/扫码
- 约 6 小时延迟，全文 + 图片代理

**目录结构确定：**
